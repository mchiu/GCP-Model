BackendServices

Backend services defines groups of backend instances and their serving capacity, which can be based on CPU or  requests per second (RPS). The backend services list groups of instances within resource views. The backend service also specifies if a health check should be performed against the available instances. Read Load Balancing for more information.

For a list of methods for this resource, see the end of this page.

Resource representations

A BackendService resource. This resource defines a group of backend virtual machines and their serving capacity.
